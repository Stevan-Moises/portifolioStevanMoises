# 📖 Sobre o Projeto

Este projeto foi desenvolvido para servir como meu portfólio profissional, apresentando minhas habilidades em desenvolvimento Front-end através de uma interface moderna, responsiva e rica em animações.

O foco principal foi criar uma experiência fluida para o usuário utilizando apenas tecnologias nativas da Web, priorizando desempenho, organização do código e boas práticas de desenvolvimento.

---

# ✨ Funcionalidades

### 🎨 Design moderno
- Dark Mode
- Glassmorphism
- Interface minimalista

### 🖱 Cursor personalizado
Cursor magnético que reage aos elementos interativos da página.

### ⚡ Scroll Reveal
Animações utilizando a API **Intersection Observer**.

### 📨 Formulário assíncrono
Integração com **Formspree** utilizando **Fetch API**, exibindo feedback visual sem recarregar a página.

### 📱 Responsividade
Compatível com smartphones, tablets e desktops.
